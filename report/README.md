# Отчет команды

В этой папке должен находиться файл отчета команды 22.

## Требования к отчету

- Формат: Word документ (.docx)
- Имя файла: `Team22_Report.docx`
- Содержание:
  - Описание шагов ручной установки
  - Скриншоты процесса установки и настройки
  - Описание автоматизации через Ansible
  - Описание процесса объединения всех серверов в единый
  - Результаты работы (метрики, данные в БД)

## Структура отчета

1. **Введение**
   - Описание задачи
   - Состав команды и распределение метрик

2. **Ручная установка**
   - Установка Midnight Commander
   - Создание каталога `/srv/team22/userX_zabbix`
   - Установка Zabbix Agent
   - Настройка метрики
   - Создание таблицы в PostgreSQL
   - Проверка работы

3. **Автоматизация через Ansible**
   - Описание Ansible роли
   - Запуск playbook
   - Проверка результатов

4. **Объединение серверов**
   - Описание процесса объединения всех серверов команды в единый
   - Настройка централизованного сбора метрик
   - Настройка единой базы данных PostgreSQL

5. **Результаты**
   - Скриншоты метрик в Zabbix
   - Данные в PostgreSQL (SELECT запросы)
   - Выводы
