# Конфигурационные файлы

Эта папка содержит примеры конфигурационных файлов для сервисов и мониторинга.

## Файлы

- `zabbix_agentd.conf.example` - пример конфигурации Zabbix Agent

## Использование

Конфигурационные файлы используются как шаблоны в Ansible роли.
Фактические конфигурации генерируются из шаблонов Jinja2 в `ansible/roles/zabbix_agent/templates/`.
